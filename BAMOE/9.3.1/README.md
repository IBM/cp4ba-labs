# IBM Business Automation Manager Open Editions

## Overview

IBM Business Automation Manager Open Editions (BAMOE) is a Business Automation foundational platform for developing, deploying, managing, and monitoring Enterprise-grade, cloud-native Business Services in hybrid cloud environments, such as Red Hat® OpenShift® and Kubernetes.

IBM BAMOE is based on Open-Source Apache KIE technologies such as Drools, jBPM, and Kogito, and provides a set of libraries, applications and tools that adapt to your Software Development Life Cycle (SDLC) and integrate with services and technologies that you are probably already using (VS Code, Git-based source code hosting services, event streaming platforms such as Apache Kafka, popular microservice monitoring and metrics technologies such as Prometheus, and AI features with Ollama and LangFlow).

## Labs

- [**Model your first Workflow and DMN with BAMOE**](Lab%2001%20-%20Explore%20the%20IBM%20BAMOE%20v9.X%20Canvas%20with%20Kubernetes.pdf): In this lab, you will:
-- Create a DMN using BAMOE Canvas;
-- Create a workflow using BAMOE Canvas;
-- Run your project using VS Code;
-- Connect your project to the BAMOE Management Console.

  ***Approximate Duration: 3 hour***

- [**Introduce AI Capabilities in your BAMOE Workflow**](Lab%2002%20-%20Exploring%20IBM%20BAMOE%20v9.X%20Developer%20Tools%20for%20VS%20Code.pdf): In this lab, you will:
-- Configure and Test Gen AI Task with Ollama;
-- VS Code - Configure and Test LangFlow AI Agent Task;
-- VS Code - Deploy and Test.

  ***Approximate Duration: 1 hours***

- [**Expose your DMN project with MCP Server**](Lab%2002%20-%20Exploring%20IBM%20BAMOE%20v9.X%20Developer%20Tools%20for%20VS%20Code.pdf): In this lab, you will:
-- Prepare a DMN project and deploy your MCP Server to Podman;
-- Build a simple agent in LangFlow to consume your DMN via MCP Server.

  ***Approximate Duration: 1 hours***
